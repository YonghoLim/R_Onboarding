6. Downloading a CSV Dataset

In this section, you will download a CSV dataset.

A CSV file is a common type of data file. CSV stands for comma-separated values.

A CSV file looks like a spreadsheet, but it is saved as plain text.

Sample Dataset

For this guide, we will use a small dataset called:

mydata.csv

It contains information about imaginary students, including:

  • Study hours
  • Sleep hours
  • Attendance
  • Quiz score
  • Major

If you are viewing this guide as a website, use this link:

Download the sample CSV dataset

TipTip

If your browser opens the CSV instead of downloading it, right-click the link and choose Save Link As…

Saving the CSV File

Save the file in your project folder.

Your folder should look like this:

r-onboarding-guide/
├── first_script.R
├── first_report.qmd
└── mydata.csv

Verifying the File Exists

In R, you can check whether the file exists:

file.exists("mydata.csv")
[1] TRUE

If R prints:

[1] TRUE

that means R found the file.

If R prints:

[1] FALSE

that means R did not find the file in the current working directory.

WarningCommon Mistake

The file name must match exactly.

These are different to R:

mydata.csv
MyData.csv
mydata (1).csv

If You Do Not Have the File Yet

If your instructor has not provided mydata.csv, you can create it yourself.

  1. Open a plain text editor.
  2. Copy the text below.
  3. Save the file as mydata.csv.
  4. Put it in your project folder.
student,study_hours,sleep_hours,attendance,quiz_score,major

Student 2,8,6,Present,91,Business
Student 3,3,8,Absent,65,Psychology

TipTip

Make sure the file name is exactly:

mydata.csv

not:

mydata.csv.txt

  2. Save it in your project folder.
  3. Run:
file.exists("mydata.csv")
  1. Make sure the result is TRUE.

Summary

In this section, you learned that:

  • A CSV file is a common data file.
  • CSV files can be opened in spreadsheet programs and R.
  • Your CSV file should be saved in your project folder.
  • file.exists() checks whether R can find the file.
